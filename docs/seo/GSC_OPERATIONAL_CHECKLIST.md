# Google Search Console Operational Checklist

These items require Search Console account access and must be completed after deployments.

## Immediate

- [ ] Submit sitemap: `https://www.floralwhispersgifts.co.ke/sitemap.xml`
- [ ] Inspect and request indexing for:
  - [ ] `/`
  - [ ] `/flower-hamper-wine-nairobi`
  - [ ] `/blog`
  - [ ] Top 10 product URLs
- [ ] Confirm selected canonical is always `www` for homepage, blog post, and product URL.

## Weekly

- [ ] Check Coverage report for new errors
- [ ] Check Page indexing report for excluded pages
- [ ] Check Enhancements report (Product snippets / structured data)
- [ ] Check top 20 queries for high impressions + low CTR pages

## Monthly

- [ ] Compare 28-day CTR and avg position against baseline
- [ ] Validate progress on:
  - homepage CTR
  - target query `nairobi flower hamper wine same day delivery`
  - product snippet CTR

## Notes

- Geo-targeting controls may vary by property type in modern Search Console. If direct geo-target is unavailable, enforce locality through content, schema, internal links, and Kenyan intent queries.
