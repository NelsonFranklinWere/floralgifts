# Weekly SEO Review Playbook

## Every Monday

1. Pull last 7 and 28 days from GSC.
2. Compare:
   - total clicks
   - total impressions
   - average CTR
   - average position
3. Review top 20 queries by impressions.
4. Flag pages with:
   - high impressions, CTR < 1.5%
   - avg position between 6 and 12
5. Plan actions:
   - rewrite meta snippets
   - add internal links
   - refresh on-page copy
6. Update `docs/seo/MASTER_TRACKER_Q2_2026.md`.
7. Append deployment outcomes in `docs/seo/SEO_RELEASE_LOG.md`.

## Monthly

1. Device-level conversion review (mobile, desktop, tablet).
2. Country-level impression quality check.
3. Top 10 page CTR and ranking movement summary.
4. Update next 4-week optimization backlog.
